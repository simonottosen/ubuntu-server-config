Waitport

You can track the waiting-time in the security across various European airports. The data is collected in real-time while the prediction model is re-trained continiously.

  • Waitport - Airport queue

Waitport API

This tool is designed to provide estimates and data of the security queue in various European airports at a user-defined point in time in the future. The tool uses Machine Learning based on historical data to estimate the queue. These links only covers the API access. For a frontend example, the Waitport link above can be visisted.
