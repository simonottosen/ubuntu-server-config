Housing market

This tool is designed to create an overview of current market changes to houses and apartments in Denmark. The tool is based on the latest price-index from nearby houses on Boliga.

  • Housing market frontpage

Electricity Price

This tool is designed to create an overview of the current price of electricity in Copenhagen for inhabitants with Modstrøm, Andel, etc. through Radius.

  • Electrity Price frontpage

CPH Security

This tool is designed to provide estimates of the security queue in the CPH airport at a user-defined point in time in the future. The tool uses Machine Learning based on historical data to estimate the queue.
